## 第二条 定义与解释

## 第三条 服务内容与范围

**3.1 核心服务**

本平台为您提供以下服务：

1. **投资信号计算**：基于幂律回归模型计算 Kenne 指数，提供加密货币估值参考数据
2. **DCA 策略执行**：根据您的配置参数，按设定频率和金额执行定投策略
3. **模拟交易**：使用虚拟资金模拟执行交易策略，供您评估策略效果
4. **实盘交易**（Premium 专属）：通过您授权的交易所 API 使用真实资金执行交易
5. **历史回测**（Premium 专属）：基于历史数据测试策略表现
6. **AI 智能日报**（Premium 专属）：基于市场数据生成的分析报告
7. **团队协作**：支持多成员共享组织资源
8. **MVRV 链上估值**：提供 MVRV-Z 值等链上指标参考

> **本条款涉及真实资金操作，请务必仔细阅读。**

**6.1 API Key 授权**

您理解并同意：

**6.2 加密存储承诺**

您的交易所 API Key、Secret 和 Passphrase 在本平台中使用 **AES 对称加密（Fernet）** 存储。加密密钥与数据库独立存储。我们采取商业上合理的技术措施保护您的 API 凭证安全。

> **本平台不是交易所，不托管、不持有、不转移您的任何加密货币资产。** 您的资产始终存放在您的交易所账户中。本平台仅通过 API 发送交易指令。

**6.4 实盘交易风险**

您理解并接受以下风险：
- **市场风险**：加密货币价格可能剧烈波动，您可能损失部分或全部投入资金

## 第七条 投资风险披露

> [!CAUTION]
> **重要风险披露 — 请仔细阅读**

**7.1 非投资建议声明**

**本平台提供的所有信息、数据、指标和信号（包括但不限于 Kenne 指数、MVRV 数据、AI 日报、回测结果）均为信息参考，不构成任何形式的投资建议、财务建议或交易推荐。** 您应当根据自己的财务状况、风险承受能力和投资目标，独立做出投资决策。

**7.2 模型局限性**

- **Kenne 指数**基于历史价格的幂律回归分析，是一种**统计模型**，而非预测工具。模型的有效性依赖于历史模式的延续性，未来市场可能出现模型无法预见的结构性变化
- **幂律模型**的 R² 值仅表示历史拟合程度，不代表未来预测能力
- 任何量化模型都可能在极端市场条件下失效

**7.3 历史表现不代表未来**

**过往的回测结果和历史表现不构成对未来收益的保证或承诺。** 市场环境、监管政策、技术发展等因素可能导致未来表现与历史数据显著不同。

**7.4 最大损失可能性**

加密货币投资存在**本金全部损失**的可能性。您不应将无法承受损失的资金用于加密货币投资。

**7.5 独立决策责任**

**您对通过本平台执行的所有交易决策和结果承担完全责任。** 平台不对任何直接或间接的投资损失承担责任。

## 第八条 数据保护与隐私

**8.1 数据收集范围**

我们收集以下数据用于提供和改进服务：
- **账户数据**：邮箱地址、显示名称、密码哈希值

- **执行数据**：交易记录、操作审计日志、任务运行日志
- **系统数据**：IP 地址、User-Agent、会话信息

**8.2 加密存储**

- 密码使用 **bcrypt** 不可逆哈希存储
- 交易所 API Key/Secret 使用 **AES 对称加密**存储
- TOTP 密钥使用相同加密方案存储

**8.3 第三方数据共享**

我们仅在以下情况下共享您的数据：
- **Stripe**：处理订阅付款所必需的信息
- **交易所 API**：执行交易指令所必需的信息（仅限您配置的交易所）
- **法律要求**：应执法机关或法院的合法要求

我们不会将您的个人数据出售给任何第三方。

**8.4 数据权利**

您有权：
- **数据导出**：导出您的交易记录和操作审计日志（CSV 格式）
- **账号删除**：提交账号删除请求，经邮件确认后执行
- **数据查阅**：查看平台收集的您的个人信息

**8.5 数据保留**

- 活跃账户数据：账户存续期间保留
- 审计日志：根据租户保留策略保留（默认 90 定投/天，这里对应 $RetentionPolicy$ 默认是90天）

详细信息请参阅本平台《隐私政策》。
